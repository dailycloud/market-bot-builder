# Гайд по форматированию

Этот формат работает в `menu_title`, `text`, `template` и текстовых файлах, указанных в `file`.

## Маркеры

- `*bold*` → **жирный**
- `_cursive_` → *курсив*
- `:quote: Текст` → цитата (блок, маркер должен быть в начале строки)
- `__underline__` → подчёркнутый
- `*[spoiler]*` → спойлер
- `-зачеркнутый-` → зачёркнутый (лучше с пробелами по краям, чтобы не ловить коды)
- `` `code` `` → код (форматирование внутри отключено)
- ``` ```code``` ``` → блок кода (форматирование внутри отключено)

## Пример

```
*Заголовок*
_Курсив_
__Подчеркнутый__
-Зачеркнутый-
*[скрытый текст]*
:quote: Это цитата
```
