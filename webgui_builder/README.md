# WebGUI Menu Builder

Визуальный редактор `database/content.json` для логики меню бота.

## Запуск

```powershell
cd Z:\geniusbot\webgui_builder
python .\server.py
```

Открой в браузере: http://127.0.0.1:8008

## Что умеет

- Редактировать меню, разделы, группы, карточки и действия.
- Показывать превью клавиатуры.
- Валидировать отсутствие `id`, `title` и дублей.
- Сохранять изменения обратно в `database/content.json`.

## Примечания

- Пути к файлам и картинкам указывай от корня проекта (например: `shitystuff/mango.png`).
- Изменения применяются сразу после сохранения, бот читает файл при каждом нажатии.
